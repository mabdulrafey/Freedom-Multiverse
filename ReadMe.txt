FREEDOM MULTIVERSE
==================

A 2D game with 3 levels. Complete each level to unlock the next.


---------------------------------
LEVEL 1 - THE MAGIC SCHOOL
---------------------------------
Genre: 2D Platformer

You play as a kid in a magic land/magic school. Find all 5 scrolls
protected by deadly snakes and a ghost. Collect all 5 scrolls to
move to the next level.

Controls:
  Left/Right Arrow  - Move
  Up Arrow          - Jump
  S                 - Cast Spell


---------------------------------
LEVEL 2 - CAPTAIN APEX VS VOID
---------------------------------
Genre: 2D Fighting

You are Captain Apex. Fight and defeat the enemy Void to advance
to the next level.

Controls:
  Left/Right Arrow  - Move
  Spacebar          - Jump
  A                 - Kick
  S                 - Punch
  D                 - Block


---------------------------------
LEVEL 3 - THE LAST STAND
---------------------------------
Genre: Top-Down Survival

You are a soldier in a battleground. Survive 5 waves of enemies.
The boss appears in the final wave. Beat all 5 waves to win.

Controls:
  Arrow Keys  - Move (Up/Down/Left/Right)
  S           - Strike with Sword


---------------------------------
CHEATS
---------------------------------
Type "skip"  - Skip the current level
Type "reset" - Restart the current level
